Challenges Faced:

While making this project, I experienced these challanges:

 Using Do-While Loops for Validation:

I found it hard to structer the loop for user input and to stop the loop after user placed the correct input
I learned that do-while loops are useful because they always run at least once and are ideal for input validation. I searched on youtube how to do them correctly

 Password Validation Logic
Creating the password validation was challenging. I first used a do while loop and many other conditions until I found the right one.

I had to learn how to use Character.isUpperCase(), Character.isDigit()

Another challenge was understanding how to correctly compare strings in Java.
Initially, I tried using ==, which did not work correctly.

I then learned that .equals() must be used. for example:
userName.equals(enteredUserName)

Debugging and Testing
At some point the program would not work as expected, especially during login and validation. It took time to trace errors and understand where the logic was failing.
 I improved this by testing each method individually and printing outputs to track program flow
